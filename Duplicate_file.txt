			Introduction to Version Control System (VCS):-
	
1)  A version control system or VCS, also know as revision control or source control system,
   is a software utility that tracks and manages changes to a filesystem.

2)  A VCS also offers collaborative utilities to share and integrate these filesystem changes to other VCS users.
	
3)  When operating at the filesystem level, a VCS will track the addition, deletion, and modification actions 
   applied to files and directories.
   
4)  A repository is a VCS term which describes when VCS is tracking a filesystem.

5)  In the scope of individual source code files, a VCS will track additions, deletions, modifications
   of the lines of text within that file.
   
6)  Popular software industry VCS options include Git, Mercurial, SVN and preforce.
